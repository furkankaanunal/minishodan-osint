OSINT Workbench

A polished browser-based dashboard for collecting and organizing passive reconnaissance results. Enter a domain, review DNS records and subdomains, document Shodan/Censys findings, track evidence, classify observations by severity, and generate a report-ready summary.

Ethical Scope

This tool is designed for passive OSINT. It does not perform brute-force attempts, exploit testing, unauthorized access, vulnerability exploitation, or invasive port scanning. It helps organize findings gathered from public sources for educational use.
