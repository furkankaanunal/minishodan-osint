Methodology

  1. Normalize the target domain.
  2. Query public DNS records through DNS-over-HTTPS.
  3. Search certificate transparency records for subdomains.
  4. Resolve discovered subdomains to identify visible IP addresses.
  5. Generate basic exposure indicators and Shodan search links.

Ethical boundary: this tool only performs passive collection from public sources. It does not test, exploit, bypass, or overload systems.
